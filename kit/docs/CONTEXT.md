# CONTEXT — Glossário canônico do domínio

> **Linguagem ubíqua (DDD):** todo doc, ADR, código e conversa usa estes termos com
> o mesmo significado. Termos em PascalCase PT-BR, em ordem alfabética. Quando um
> termo é decidido numa sessão de grilling (`grill-with-docs`), entra aqui apontando
> para o ADR onde a decisão foi cristalizada.

## Convenção de idioma
- Termos do domínio são canônicos em PT-BR.
- Exceções em código (ex.: nome de tabela/tipo em inglês) são documentadas no próprio termo.
